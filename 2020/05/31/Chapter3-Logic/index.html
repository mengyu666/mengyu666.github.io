Lecture 6 Set Theory Defining Sets

Sets

A set is a collection of definite, distinct objects.

The basic operation on a set

Given some set S and some object x we can ask the question:“is x an element of S?”

  • $x ∈ S$ when x is an element of the set S
  • $x\notin S$ when x is not an element of the set S

we cannot ask where x is in S (at the front, on top, …)or how many times x occurs in S, …

Defining Sets

By enumeration

S :={ 0, 2, 4, 6, 8, 10 }

  • Enumeration: just list all the set’s elements

  • The order of the elements doesn’t matter

  • This only works for finite sets

    comprehension

    T := { x ∈ S | x > 5 • x }

  • For all elements x in the (already defined) set S

  • Filter to those that satisfy x > 5

  • And keep the value x

注意这四个公式：

  1. $φ ↔ ψ$ and $(φ → ψ) ∧ (ψ → φ).$
  2. $φ → ψ$ and $¬φ ∨ ψ.$
    3a. $φ ∧ ψ$ and $¬(¬φ ∨ ¬ψ).$
    3b. $φ ∨ ψ$ and $¬(¬φ ∧ ¬ψ).$

举个例子：
$(X\leq 5)\rightarrow (X^{2}< 10)\cdot X+1$
可以转换为：¬$ (X\leq 5)$∨ $(X^{2}< 10)$

The empty set

  • The empty set is the set with no elements
  • Usually named ∅, and enumerated as { }

Lecture 7 Syllogistic Reasoning

Syllogisms

definition

A syllogism is a inference with specific characteristics:

  • Just two premises
  • The premises and conclusion can have only the following forms:
    1 All A are B
    2 Some A are B
    3 All A are not B (i.e. No A is B)
    4 Some A are not B (i.e. Not all A are B)
    where A and B are predicates, that is, they represent collection of
    objects.
  • The inference involves just three predicates.

Syllogistic Reasoning 三段论推理

Syllogistic reasoning allows us to check the validity of a given syllogistic
inference.
The process is:

  • We interpret the three predicates as three sets.
  • We interpret each of the premises in turn as applying constraints on the membership of these sets.
  • Having processed all the premises, we evaluate the conclusion to see if it is supported in all of the scenarios that result.
    We use Venn diagrams to help us represent the reasoning process.

Process the premises:
i. Crossing out - Universal step. A premise that starts with all/no means you cross out certain ‘forbidden’ regions in the diagram.
ii. Marking in - Existential step. A premise that starts with some/not-all means you mark certain areas as inhabited (if they haven’t been crossed out).

Counting Syllogistic Forms: Moods

The mood of a syllogistic inference is a label based on the forms of the two premises and conclusion.

How many kinds of syllogistic inferences are there?

There are three statements in a syllogistic inference

Lecture 9 Relations

Using sets to define relations

By a relation we mean a meaningful link between people, things,objects, whatever.

Ordered Pairs

An ordered pair is an object of the form (a, b) where a and b are some already-existing objects.

Equality:
We define (a, b) = (c, d) if any only if a = c ∧ b = d.
For a given pair (a, b) we refer to a and b respectively as the first and second elements of the pair
Ordering: Note that, in general, (a, b) is different from (b, a).
In the special case where (a, b) = (b, a) we can deduce from thedefinition of equality that a = b ∧ b = a.

(Ordered) Tuples （有序）元组

We can extend the definition of a pair to define an (ordered) triple asbeing an object of the form (a, b, c) …
We can generalise this by defining an n-tuple as being a sequence of n objects of the form (x1, x2, . . . , xn)
We get at an element of an n-tuple by using a projection function
πi(x1, x2, . . . , xn) , xi for i ∈ {1, . . . , n}
In particular, for any pair (a, b)
π1(a, b) = a
π2(a, b) = b

  • 元组可能包含同一元素的多个实例
  • 元组元素是有序的
  • 元组具有有限数量的元素，而集合或多集可能具有无限数量的元素

Cartesian Product

We define:
The Cartesian Product of two sets A and B is the set of all pairs of the form (a, b), for every a ∈ A and b ∈ B.
Notation: the Cartesian product of A and B is the set A × B
A × B:={a ∈ A, b ∈ B • (a, b)}

Defining a relation in terms of sets

We can now give a definition of relations in terms of set theory…
Given two sets A and B we define:
a binary relation between A and B is a subset of A × B.

Similarly, given three sets A, B and C,
a ternary relation between them is a subset of A × B × C

… given n sets S1, S2, . . . , Sn,
an n-ary relation between them is a subset of S1 × S2 × . . . × Sn

That is: an n-ary relation is a set
of n-tuples.

Relational Operators

Composing two relations

Given three sets A, B and C, and two binary relations R ⊆ (A × B) and S ⊆ (B × C) we define the composition of R and S as the relation R ; S , {a ∈ A, b ∈ B, c ∈ C | (a, b) ∈ R,(b, c) ∈ S • (a, c)}

Note that (R ; S) ⊆ (A × C)

Notation:
We use the notation as R ; S, “R then S”
Also written as S ◦ R, “S after R” (not standard! opposite in textbook)

The inverse of a relation

Given two sets A and B, and a binary relation R ⊆ (A × B) we define the
inverse of R as the relation
$R^{-1}$ , {a ∈ A, b ∈ B | (a, b) ∈ R • (b, a)}
Note that $R^{-1}$ ⊆ (B × A)
Notation:
This is usually written as $R^{-1}$Also written as R∼ or R˘ (unusual: textbook)

Examples:
The inverse of {(1, 2),(2, 3)} is {(2, 1),(3, 2)}
Numbers: the inverse of the less-than relation is just the greater-thanrelation
Tudors: the inverse of is-a-parent-of is just is-a-child-of

Lecture 10 Representing and Classifying Relations

Some notation

  • Given some n-ary relation R, we denote membership of that relation by saying something like (x1, x2, . . . , xn) ∈ R
  • Sometimes it is more convenient to use prefix notation: if we want toexpress that x1, x2, . . . , xn are related by the relation R we might just write R(x1, x2, . . . , xn)
  • When we are dealing with a binary relation, we might choose to write the relation symbol infix, as in xRy.
    The usual integer relations (<, >, etc.) are examples of binary relations that are typically written using infix notation.

Representing relations using an adjacency matrix

Adjacency Matrix

One way of representing a (finite) binary relation is as an adjacency matrix.
Each entry in the adjacency matrix is a boolean value.
Given any finite binary relation R ⊆ A × B, we can construct its adjacency matrix MR as follows:
We index the rows and columns of the matrix by the elements of A and B respectively (assumed to be in some fixed order).
For any (a, b) ∈ (A × B) we set the entry MR for row a and column b to true iff (a, b) ∈ R

Classifying Relations

reflexive

If A is a set, then the relation I = {a ∈ A • (a, a)} is called the identity relation on A.
That is, the identity relation just maps every object to itself.

Any binary relation R that has the property that I ⊆ R called a reflexive relation.
That is, a reflexive relation at least maps every object to itself, and maybe to some other objects too.

Any binary relation R that has the property that I ∩ R = ∅ called an irreflexive relation.
That is, an irreflexive relation never maps any object to itself.
如果A是一个集合，那么关系I={A∈A•（A，A）}称为A上的恒等关系。
也就是说，身份关系只是将每个对象映射到自身。

任何具有I⊆R所称自反关系性质的二元关系R。
也就是说，一个自反关系至少将每个对象映射到它自己，也可能映射到其他一些对象。

任何具有I∩R = ∅的性质的二元关系R称为不可伸缩关系。
也就是说，不可伸缩的关系从不将任何对象映射到自身。

symmetric(对称)

Any binary relation R that has the property that $R^{-1}$ ⊆ R is called symmetric.
i.e. whenever it contains the pair (a, b) it also contains the pair (b, a).

An antisymmetric relation is a binary relation with the property that if it ever contains both (a, b) and (b, a) then we must have a = b.

i.e. an antisymmetric relation can be reflexive, but is not otherwise symmetric.

Aside: A relation that is both antisymmetric and also irreflexive is called asymmetric.

reflexive: for all x, R(x, x)
irreflexive: for all x, ¬R(x, x)
symmetric: for all x, y, R(x, y) → R(y, x)
antisymmetric: for all x, y, (R(x, y) ∧ R(y, x)) → x = y
asymmetric: for all x, y, R(x, y) → ¬R(y, x)

Transitive Relations

Any binary relation R that has the property that whenever (x, y) ∈ Rand (y, z) ∈ R then (x, z) ∈ R is called transitive.

Any binary relation that is reflexive, transitive and also antisymmetric is called a partial order. (no cycles)

If, in addition, for every two objects x, y we have either (x, y) ∈ R or (y, x) ∈ R, then we call it a total order. (a chain)

Any binary relation that is reflexive, transitive and also symmetric is called an equivalence relation.

Equivalence relations and partitions

Given any set S, a partition of S is a set of subsets that are collectively exhaustive and mutually exclusive.
That is, sets S1, . . . , Sn are a partition of S if:

  • (S1 ∪ . . . ∪ Sn) = S [collectively exhaustive]
  • for any sets Si and Sj , if i not equal j then Si ∩ Sj = ∅

Any equivalence relation over some set S automatically partitions S into a set of equivalence classes.
