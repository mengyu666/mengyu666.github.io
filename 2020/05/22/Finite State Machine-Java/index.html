Finite State Machine

Author: master Date: May 22, 2020  11:08:59

Introduction

the 5 things we need for a Finite State Machine

• Q: a finite set pf states
• Σ: a finite alphabet set
• q0: (q0 ∈ Q): an element of the states (Q) which states the start state
• F: (F ⊆ Q): the Final state
• δ:Q.Σ → Q: transition function-a set of maps from states and inputs into states

M = ( S，Σ ，δ，s0，F )
 S：有穷状态集
 Σ：输入字母表，即输入符号集合。假设ε不是 Σ中的元素
 δ：将S×Σ映射到S的转换函数。s∈S, a∈Σ, δ(s,a)表示从状态s出发，沿着标记为a的边所能到达的状态。
 s0：开始状态 (或初始状态)，s0∈S
 F：接收状态（或终止状态）集合，F⊆ S

Example

• Q: {q0, q1, q2} //3 states here
• Σ: {a, b} //only a and b in the language
• q0: (q0 ∈ Q): q0 //q0 is an element of the states (Q)
• F: (F ⊆ Q): q2 //q2 is the Final state

• δ: Q.Σ → Q:
q0.a → q1 // if we are in state q0 and we get an a we move to state q1
q0.b → q2 // if we are in state q0 and we get an b we move to state q2
q1.a → q0
q1.b → q2
q2.a → q2
q2.b → q2

Transition Table

state a b
(start)q0 q1 q2
q1 q0 q2
(final)q2 q2 q2

NFA(不确定的有穷自动机)和DFA(确定的有穷自动机)

DFA与NFA的区别:DFA的每一次输入只对应一个结果，而NFA的依次输入可能对应多个结果，形成一个结果集

NFA

state 0 1
(start) q0 q0, q1 q0, q2
q1 q3
q2 q3
(final) q3

DFA

state 0 1
(start)q0 q1 q2
q1 q3 q2
q2 q1 q4
q3 q3 q2
(final)q4 q1 q4
