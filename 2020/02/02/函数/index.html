函数的定义

函数定义的语法

数据类型 函数名（形式参数表）
{
函数体//执行语句
}
注意
1.形式参数表可以是空的，也可以有多个形参，形参间用逗号隔开，不管有没有参数，函数名后圆括号必须有。形参必须要有类型说明。作用是实现主调函数与被调函数之间的的关系。
2.函数不允许嵌套定义，但允许嵌套使用

如果函数定义在主函数前，函数不必声明

函数定义的形式

1.无参函数 没有参数传递的函数 函数类型说明为void
2.有参函数 有传递参数的函数 一般要带回函数值
3.空函数 在大型程序中，空函数用于扩充函数功能
函数名（）
{ }

函数的声明与调用

函数的声明

格式：
类型说明符 被调函数名（含类型说明的形式参数表）；

在所有函数定义前声明，该函数可应用于本程序文件
在某个主调函数中声明，该函数只能应用于该主调函数

函数的调用

函数名（实参列表）
实参列表中应给出与函数原型形参个数相同，类型相符的实参。
函数调用可以作为一条语句，这时函数可以没有返回值
函数调用可以出现在表达式中，这时候要有一个明确的返回值

实参：在主调函数中的参数称为实参，实参一般具有具体的值。
形参：

函数的返回值

两种：

  • 有返回值的函数
    return(表达式）；返回值的类型为它所在函数的函数类型
  • 无返回值函数
    可以没有return语句，直接利用函数体的右括号“}”作为返回值的函数返回
    return；直接return，没有表达式

    函数的传值调用

    函数的调用过程实际上是对栈空的操作过程，因为调用函数是使用栈空间来保存信息的。函数在返回时，如果有返回值，则将它保存在临时变量中。然后回复主调函数的运行状态，释放被调函数的栈空间，按其返回地址返回到调用函数。

传值调用（没有取址符“&”）

将实参的数值传递给形参，即将实参值拷贝一个副本存放在被调函数的栈区中，在被调函数中，形参值可以改变，但不影响主调函数的实参值。参数传递方向只是从实参到形参，简称单向值传递。

###传址调用（有取址符“&”）
将实参函数的地址值传递给形参，这时形参是指针，即让形参的指针指向实参地址。在被调函数中，形参值改变，影响主调函数的实参值。

注意
将数组名作为参数传给函数，实际上是把数组的地址传给函数

全局变量、局部变量、及他们的作用域

全局变量

  • 定义在函数外部没有被花括号括起来的变量称为全局变量。全局变量的作用域是从变量定义的位置开始到文件结束。
  • 全局变量在程序执行的过程中一直占用着内存单元。
  • 全局变量若在定义时没有赋值，其默认值为0。

    局部变量

    局部变量的储存空间是临时分配的，当函数执行完毕，局部变量的空间就会被释放，其中的值无法保留到下次使用。
    在不同的函数中变量名可以相同，它们分别代表不同的对象，在内存中占据不同的单元，互不干扰。
    一个局部变量和一个全局变量是可以重名的，在相同的作用域内局部变量有效时而全局变量无效。
    主函数main中定义的变量也是局部变量
    全局变量数组初始全部为0，局部变量值是随机的，要初始化初值，局部变量收栈空间大小限制（ps：局部变量的数组不能开很大，全局变量随便）
