封装

Author: master Date: June 15, 2020  18:46:03

Java 封装

指一种将抽象性函式接口的实现细节部分包装、隐藏起来的方法。

封装可以被认为是一个保护屏障，防止该类的代码和数据被外部类定义的代码随机访问。

要访问该类的代码和数据，必须通过严格的接口控制。

封装的优点

  1. 良好的封装能够减少耦合。

  2. 类内部的结构可以自由修改。

  3. 可以对成员变量进行更精确的控制。

  4. 隐藏信息，实现细节。

实现Java封装的步骤

  1. 修改属性的可见性来限制对属性的访问.

2.对每个值属性提供对外的公共方法访问，即setter和getter.
