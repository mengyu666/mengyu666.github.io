多态

多态是同一个行为具有多个不同表现形式或形态的能力。

多态存在的三个必要条件

  • 继承
  • 重写
  • 父类引用指向子类对象

    多态变量

    Java的对象变量是多态的，它们能保存不止一种类型的对象
    即既可以保存声明类型的对象，又可以保存声明类型的子类的对象

    对象变量的类型

  • 声明类型(静态类型): 字面上看出来的类型
  • 动态类型: 程序运行到这里时，它里面管理的对象的类型

    造型

    把一个类型的对象赋给另一个类型的对象的过程
    如: 子类的对象赋给父类的变量 (向上造型)
    [不能说子类的对象赋给父类的变量对象☺]

当使用多态方式调用方法时，首先检查父类中是否有该方法，如果没有，则编译错误；如果有，再去调用子类的同名方法。

WHY?

虚函数

指使用多态，要调用某一个函数，父类和子类都要有该函数

说明：
实例中，实例化了两个 Dog对象：一个使用 Animal 引用 animal ，另一个使用 Dog 引用 dog 。

因为 animal 是 Animal 的引用，所以调用 animal 的 move() 方法时，编译器会去 Animal 类查找 move() 方法 。

在编译的时候，编译器使用 Animal 类中的 move() 方法验证该语句， 但是在运行的时候，Java虚拟机(JVM)调用的是 Dog类中的 move() 方法。

当调用dog.move() 时，编译器在编译时会在 Dog 类中找到 move()，执行过程 JVM 就调用 Dog 类的 move()。
