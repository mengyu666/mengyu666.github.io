CSS基本语法

Author: master Date: June 19, 2020  15:37:00 Category: 前端

< style>< /style> 是特殊的区域，要用CSS语法

注释

CSS 基本语法：

  • 选择器：通过选择器可以选中页面中的指定元素
    如p的作用就是选中页面中的所有p元素
  • 声明块：通过声明块指定要为元素设置的样式(就是大括号中内容)
    • 声明块由一个个声明组成
    • 声明是一个名值对结构，一个样式名对应一个样式值 格式： 名:值；
