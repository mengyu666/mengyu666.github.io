样式的继承

样式的继承，我们为一个元素设置的样式同时也会应用到它的后代元素上

  • 继承是发生在祖先后后代之间的
  • 继承的设计是为了方便我们的开发，利用继承我们可以将一些通用的样式统一设置到共同的祖先元素上，这样只需设置一次即可让所有的元素都具有该样式
    注意：
    并不是所有的样式都会被继承：比如 背景相关的，布局相关等的这些样式都不会被继承。 效果展示如下：
